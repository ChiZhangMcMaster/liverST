
# liverST

<!-- badges: start -->
<!-- badges: end -->

An R package providing a reproducible spatial transcriptomics analysis
pipeline built on Seurat and related ecosystems, designed for interactive
and web-based applications.

## Installation

You can install the development version of spatialPipelineR like so:

``` r
install.packages("remotes")
remotes::install_github("ChiZhangMcMaster/liverST")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(spatialPipelineR)
## basic example code
```
